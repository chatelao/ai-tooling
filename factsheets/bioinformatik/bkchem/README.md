# Factsheet: BKChem

## Gruppe: Bioinformatik

## Zweck: BKChem ist ein Werkzeug für

- das Zeichnen chemischer Strukturen in 2D.
- den Export von Strukturen in verschiedene Formate (SVG, PDF, PNG, CDML).
- die Bearbeitung von Moleküleigenschaften.

## Reifegrad

Veraltet (Inkompatibel mit Python 3.12)

## Technische Schulden

Hoch

## Erwartetes Lebensende

Projekt eingestellt

## Installation (Ubuntu 24.04)

```bash
sudo apt update
sudo apt install bkchem
```

## Beispieldaten

Die folgenden Beispieldaten befinden sich im Ordner `examples/`:

- `benzene.mol` - Eine Model-Datei für Benzol.
- `water.cdml` - Eine BKChem-native XML-Datei für Wasser.

## Validierung

Starten Sie BKChem über die Kommandozeile oder das Anwendungsmenü:

```bash
bkchem
```
