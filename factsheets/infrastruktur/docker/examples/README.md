# Docker Beispiele

Dieser Ordner enthält typische Konfigurationsdateien für Docker-Projekte.

- `Dockerfile`: Beispiel für eine Python Flask Anwendung.
- `.dockerignore`: Ausschlussregeln für den Build-Kontext.
- `entrypoint.sh`: Initialisierungsskript.
- `app.py`: Minimaler Flask-Service.
- `nginx.conf`: Konfiguration für einen Nginx Reverse-Proxy.
