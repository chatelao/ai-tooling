# Docker Compose Beispiele

Dieser Ordner enthält verschiedene `docker-compose.yml` Konfigurationen.

- `docker-compose.yml`: Standard Stack mit Web, DB und Redis.
- `docker-compose.gpu.yml`: Konfiguration für GPU-Unterstützung.
- `.env.example`: Beispiel für Umgebungsvariablen.
- `docker-compose.test.yml`: Setup für automatisierte Tests.
- `monitoring.yml`: Beispiel für Monitoring-Dienste.
