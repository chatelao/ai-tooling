# Kubernetes Beispiele

Dieser Ordner enthält grundlegende Kubernetes-Manifeste.

- `deployment.yaml`: Definiert ein Deployment mit 3 Repliken einer Flask-App.
- `service.yaml`: Exponiert das Deployment über einen LoadBalancer.
- `configmap.yaml`: Beispiel für Konfigurationsdaten.
- `ingress.yaml`: Konfiguration für einen Ingress-Controller.
- `gpu-pod.yaml`: Beispiel für einen Pod, der eine GPU anfordert.
