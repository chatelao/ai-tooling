# Factsheet: React

## Gruppe: App-Entwicklung

## Zweck

React ist eine JavaScript-Bibliothek zum Erstellen von Benutzeroberflächen,
insbesondere für Single-Page-Webanwendungen.


## Reifegrad

Stabil

## Technische Schulden

Gering

## Erwartetes Lebensende

Kein EOL bekannt

## Installation (Ubuntu 24.04)

React wird üblicherweise über `npm` oder `yarn` in ein Projekt eingebunden.

```bash
npm install react react-dom
```

Oder ein neues Projekt erstellen:

```bash
npx create-react-app my-app
```

## Beispieldaten

Die folgenden Beispieldaten befinden sich im Ordner `examples/`:

- `App.js` (Hauptkomponente)
- `index.js` (Einstiegspunkt)
- `package.json` (Abhängigkeiten)
- `App.css` (Styles)

## Validierung

In einem React-Projekt die Version in `package.json` prüfen oder:

```bash
npm list react
```
