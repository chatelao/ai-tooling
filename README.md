# KI-Agenten Werkzeuge & Quellen

Dieses Repository dient der Dokumentation und Verwaltung von Werkzeugen und
Datenquellen, die für den Einsatz mit KI-Agenten auf Ubuntu 24.04 (Noble)
optimiert sind. Es stellt eine strukturierte Basis dar, um die Interaktion
zwischen Mensch, KI und technischem System zu verbessern.

## Zentrale Dokumente

- **[GEMINI.md](GEMINI.md)**: Das zentrale Strategiedokument. Es beschreibt den
  Prozess der Werkzeugfindung (Discovery), Dokumentation und Wartung.
- **[TOOLS.md](TOOLS.md)**: Eine nach Gruppen sortierte Liste von
  Software-Werkzeugen inklusive Zweck, Installationsbefehlen und
  Referenzhandbüchern.
- **[DATASOURCES.md](DATASOURCES.md)**: Eine Übersicht über relevante
  Datenquellen, deren Zweck und die Art des Zugriffs.
- **[Factsheets](factsheets/README.md)**: Hierarchisch strukturierte
  Zusammenfassungen aller verfügbaren Werkzeuge nach Gruppen geordnet.

## Zweck des Repositories

Das Ziel ist es, KI-Agenten eine klare und maschinenlesbare Übersicht über die
verfügbaren Ressourcen im System zu geben. Dies umfasst:

- Automatisierte Identifikation installierter Pakete.
- Standardisierte Dokumentation in Tabellenform.
- Bereitstellung von Installationsanweisungen für eine reproduzierbare
  Umgebung.

---
*Hinweis: Diese Dokumentation ist primär für die Verarbeitung durch LLMs und
KI-Agenten optimiert, bleibt aber für menschliche Nutzer gut lesbar.*
